Without proper shutdown, the pc's performance decreases.

In Windows 11, any shutdown method doesnot shutdown your system properly. To fix this you have two solutions:

  • Restart your system everyday
  • Use Command Prompt everytime you shutdown your system.

    • Shutdown commands for command prompt:

  • shutdown/s - Shutdown the system (after 30s)
  • shutdown/r - Restart the system (after 30s)
  • shutdown/h - Hibernate the system (after 30s)
  • shutdown/p - Instant shutdown
  • shutdown/a - Abort the system shutdown
